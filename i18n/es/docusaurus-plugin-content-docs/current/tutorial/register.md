# Crear cuenta

¡Bienvenido a **MyAccess**! Únete hoy mismo para empezar a usar tu identidad digital y desbloquear recompensas exclusivas.

![Registrarse en MyAccess](img/register.png)

## Formulario de registro

Para crear tu cuenta, completa los siguientes campos:

### Correo electrónico

Introduce tu correo electrónico. Este se utilizará para la verificación de la cuenta y las notificaciones.

[tú@ejemplo.com](mailto:tú@ejemplo.com)

### Nombre de usuario

Elige un nombre de usuario único para tu cuenta. Esta será tu identidad en la plataforma.

### Contraseña

Crea una contraseña segura para proteger tu cuenta.

**Requisitos:**
- Al menos 8 caracteres
- Incluye mayúsculas y minúsculas
- Incluye al menos un número
- Considera usar caracteres especiales (p. ej., `!`, `@`, `#`, `$`, `%`, `^`, `&`, `*`)

### Confirmar contraseña

Vuelve a ingresar tu contraseña para confirmarla. Esto garantiza que la hayas ingresado correctamente.

### Número de teléfono (opcional)

Ingresa tu número de teléfono si deseas recibir notificaciones por SMS o autenticación de dos factores.

+1 (555) 123-4567

### Términos de servicio

☑ Acepto los [Términos de servicio](#) y la [Política de privacidad](#).

## Crear cuenta

Haz clic en el botón **Crear cuenta** para completar el proceso de registro.

> ¿Ya tienes una cuenta? [Iniciar sesión](#)

## Consejos para una contraseña segura

- Evite palabras o secuencias comunes como "contraseña123".
- Combine letras, números y símbolos.
- Considere usar un administrador de contraseñas para generar y almacenar contraseñas seguras.
- Actualice su contraseña regularmente para mayor seguridad.
